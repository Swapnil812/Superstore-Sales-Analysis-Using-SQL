Superstore Sales Analysis Using SQL
Project Overview

This project focuses on analyzing the Superstore sales dataset using SQL. The main objective of the project is to apply important SQL concepts such as Subqueries, Common Table Expressions (CTEs), Window Functions, JOINs, and Aggregation Functions to solve real-world business problems and generate meaningful insights from the data.

The dataset was first imported into MySQL Workbench and then normalized into separate tables including customers, products, and orders. Various SQL queries were written to analyze customer behavior, sales performance, product profitability, and regional sales trends.

Objectives
Import and manage sales data using MySQL
Normalize raw data into structured tables
Perform sales analysis using SQL queries
Apply Subqueries for advanced filtering
Use CTEs for cleaner and reusable queries
Implement Window Functions such as ROW_NUMBER() and RANK()
Solve business-related analytical queries
Generate insights from customer and sales data
